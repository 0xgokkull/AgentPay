# AgentPay: Autonomous Financial Protocol

AgentPay is a premium, high-performance interface for autonomous financial agents, featuring cross-chain routing, vault management, and real-time protocol monitoring.

## 🚀 Key Features

- **Control Center Dashboard**: Real-time system monitor with live metrics (latency, network status, block height).
- **Autonomous Agent Console**: Advanced prompt engineering interface for Payment, Registration, and Treasury agents.
- **Chain Proof Verification**: Integrated on-chain proof cards with direct links to the **Blockscout Westend Testnet** explorer.
- **Global State Management**: Persistent execution status across sessions using Zustand.
- **Premium UI/UX**: Professional glassmorphic design system powered by Framer Motion, Tailwind CSS, and Lucide React.

## 🛠 Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **State**: Zustand
- **Icons**: Lucide React
- **Agent Backend**: Groq LLAMA-3 for intelligent planning

## 🏗 Getting Started

First, install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:5003](http://localhost:5003) to access the Control Center.

## 🚢 Deployment

Optimized for Vercel. Ensure your environmental variables (`groq`, `GROQ_MODEL`) are configured in the Vercel Dashboard.
